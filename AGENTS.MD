# Project Instructions

- Always use tanstack start server functions.
  - Never make scripts that bypass the framework
  - never make REST api endpoints unless they are exposing existing server functions
- always use the Drizzle ORM. Never raw SQL queries
- always use TypeScript and ESM.
- always use static imports. Never dynamically import modules
- Always use Tailwind CSS and ShadCN components. Install existing components over making your own.
- Do not duplicate or copy/paste functionality across multiple components. Abstract them into utility/lib files or separate components
